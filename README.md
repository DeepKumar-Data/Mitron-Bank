# Mitron Bank Data Analysis

## Description
This project was created as part of the Codebasics Resume Project Challenge (Jan 2024). The main objective is to provide actionable insights to the Product Strategy Team in the banking domain by analyzing customer demographics and spending patterns.

## Features
- **Demographic Analysis**: Insights into the customer base segmented by age, gender, marital status, city, and occupation.
- **Spending Insights**: Detailed breakdown of expenditures across various categories (Entertainment, Apparel, Electronics, etc.) and preferred payment methods (UPI, Credit Card, Debit Card, Net Banking).
- **Trend Tracking**: Analysis of monthly spending behavior over a 6-month period (May to October).
- **Interactive Dashboards**: Dedicated report pages for Overview, Demographics, Spend Insights, and Trends.

## Tech Stack
- **Power BI**: Data modeling, visualization, and interactive dashboard creation.

## Folder Notes
- **`Datasets/`**: Contains the raw CSV data files used for analysis.
  - `dim_customers.csv`: Customer demographic data including income, city, and occupation.
  - `fact_spends.csv`: Transaction data showing monthly spending by category and payment type.
- **`Images/`**: Includes exported screenshots of the Power BI dashboard pages (Overview, Demographics, Spend Insights, Trends) and the data model structure.
- **`meta_data.txt`**: A data dictionary detailing the columns and values found in the Datasets.
- **`Problem Statement.pdf`**: The official problem description and requirements for the project challenge.
- **`Mitron Bank.pbix`**: The complete Power BI project file containing all queries, models, and visuals.
- **Root Images (`.png` files)**: Visual assets including landing page previews.
